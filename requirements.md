Overview:
In the Object-oriented assignment, you built the class definitions for the student - advising application.  In this assignment, you will create two data files, and create an user interface to access and manage the data. 
To create the data files, you may use a generative tool (e.g. gemini.google.com) with a prompt "create a csv file with 10 student records with name, id, major, and gpa as the columns."
The user interface will use the class definitions to instantiate a the list of students and advising schedules. 
The user interface will allow the user to update student data, and add new advising data.
The system should also, save the data to the csv files.
